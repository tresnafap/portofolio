Tutorial Membuat Canvas

APASIH CANVAS ITU?

Canvas dalam konteks HTML adalah elemen yang digunakan untuk menggambar grafik dan manipulasi gambar secara dinamis menggunakan JavaScript. Elemen < canvas > memungkinkan kamu untuk menggambar secara langsung ke dalam halaman web, membuat grafik, atau bahkan mengedit gambar.


Markup canvas <canvas>

<canvas id="kanvas" width="220" height="220"></canvas>

Langkah-Langkah Membuat Canvas di HTML :

1. Membuat Struktur Dasar HTML

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tutorial Kanvas</title>
</head>

<body>

2. Menambahkan Elemen <canvas>

<canvas id="kanvas" width="220" height="220"></canvas>

3. Menambahkan CSS untuk Styling

Buat CSS untuk menata tampilan elemen-elemen yang telah dibuat agar terlihat rapi. Dapat menggunakan properti seperti margin, padding, dan background-color untuk mempercantik tampilan.

4. Menambahkan Elemen Gambar

Di dalam elemen <body>, tambahkan elemen <img> yang akan digunakan sebagai sumber gambar yang akan digambar pada canvas.

<img id="gambar" src="assets/images/namafile.png" alt="Gambar" width="220" height="220">

5. Menambahkan JavaScript

<script>
  var imgLoaded = false;
  var img = document.getElementById("gambar");

  img.onload = function() {
    imgLoaded = true; // Menandakan gambar sudah dimuat
  }

  function kanvas() {
    var c = document.getElementById("kanvas");
    var ctx = c.getContext("2d"); // Mengambil konteks canvas

6. Menggambar di Canvas

function draw() {
  ctx.fillStyle = "blue"; // Set warna
  ctx.beginPath(); // Mulai menggambar
  ctx.arc(200, 200, 50, 0, Math.PI * 2); // Buat lingkaran
  ctx.fill(); // Isi lingkaran
}

7. Membersihkan Canvas (Opsional)

Jika ingin menghapus gambar atau membersihkan canvas, gunakan clearRect untuk menghapus konten di dalam area tertentu:

  function clearCanvas() {
    ctx.clearRect(0, 0, c.width, c.height);
  }
</script>

6. Finish
