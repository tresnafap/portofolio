Tutorial Membuat Form

Dalam membuat sebuah form pada HTML,tentunya membutuhkan elemen-elemen agar form tertata dengan rapi sesuai keinginan. Berikut elemen-elemen yang digunakan untuk membuat form sederhana pada HTML :


<form> <label> <input> <button> <textarea> <option> <select>

Contoh Form Login & Register :

Login

Register


Penjelasan Elemen :

<form> : Elemen ini adalah wadah untuk semua elemen input dalam form. Ini digunakan untuk mengirim data pengguna ke server.

<label> : Elemen ini digunakan untuk memberikan deskripsi atau penjelasan pada elemen input.

<input> : Elemen ini digunakan untuk mendefinisikan field atau untuk berbagai jenis input seperti teks, angka, email, password, dll.

<button> : Elemen ini digunakan untuk membuat tombol yang melakukan tindakan, seperti mengirim form ke server.


Langkah-Langkah Membuat FOrm Login Sederhana :

1. Membuat Struktur Dasar HTML

Sebelum membuat tabel, pastikan kamu sudah menulis struktur HTML yang benar. Dokumen HTML dimulai dengan deklarasi <!DOCTYPE html>, kemudian diikuti dengan elemen <html>, <head>, dan <body>.


2. Menambahkan Tag <form> untuk Membuat Form

Gunakan elemen <form> sebagai wadah utama yang berisi input dari pengguna. Tentukan atribut seperti action dan method untuk mengarahkan data ke server ketika form dikirim.


3. Membuat Input Field untuk Username dan Password

Gunakan elemen <input> untuk membuat field input seperti username dan password. Pastikan menambahkan atribut type sesuai jenis data, seperti type="text" untuk teks dan type="password" untuk kata sandi.


4. Menambahkan Tombol Submit dan Reset

Gunakan elemen <input type="submit"> untuk membuat tombol "Login" atau "Register" dan <input type="reset"> untuk mengatur ulang field input. Tambahkan juga gaya CSS agar tombol terlihat menarik dan responsif.


5. Menambahkan Tombol untuk Berpindah Antara Login dan Register

Tambahkan tombol tambahan untuk beralih antara form login dan register, seperti <input type="button">. Buat aksi ini dinamis dengan menggunakan fungsi Javascript yang menampilkan atau menyembunyikan form berdasarkan klik pengguna.


6. Menambahkan Fungsi Javascript untuk Mengalihkan Form

Buat dua fungsi Javascript: showRegisterForm() untuk menampilkan form register dan showLoginForm() untuk kembali ke form login. Gunakan metode style.display untuk mengubah visibilitas form.


7. Membuat CSS untuk Desain Form yang Responsif

Gunakan CSS untuk memberikan tampilan yang menarik dan responsif. Tentukan ukuran, warna, padding, dan margin untuk form, input, dan tombol agar form terlihat rapi di berbagai ukuran layar.


8. Finishing

Sesuaikan hasil akhir table sesuai keinginan mu.
