Tutorial Deploy Website ke GitHub

Deploy website ke GitHub merupakan langkah penting dalam menampilkan dan mengelola proyek web secara online. Dengan GitHub Pages, Anda dapat mempublikasikan situs web secara gratis dan mudah diakses oleh siapa saja. Di halaman ini, saya ingin berbagi langkah-langkah sederhana yang sering saya gunakan untuk mengunggah dan meng-update website melalui GitHub. Mulai dari membuat repositori, menjalankan perintah-perintah Git dasar, hingga mengonfigurasi pengaturan GitHub Pages


Langkah 1 : Membuat Repositori di GitHub

Langkah pertama adalah membuat repositori di GitHub. Ikuti langkah-langkah berikut:

  1. Masuk ke akun GitHub Anda dan klik tombol New di halaman utama untuk membuat repositori baru.
  2. Klik tombol "New" di pojok kanan atas.
  3. Isi nama repositori dan dan pastikan repositori disetel sebagai "Public".
  4. Klik tombol "Create repository".

Langkah 2 : Menyiapkan File Website di Folder Lokal

Setelah repositori dibuat, Anda perlu menyiapkan file HTML dan CSS untuk website Anda.

  1. Pastikan semua file HTML, CSS, JavaScript, dan aset lainnya sudah lengkap dalam satu folder di komputer Anda.
  2. Di dalam folder proyek ini, buat file bernama index.html karena GitHub Pages akan menjadikan file ini sebagai halaman utama.

Struktur folder Anda harus seperti ini:

                    my-website/
                    ├── index.html
                    └── style.css

Langkah 3 : Push Proyek ke GitHub

Setelah proyek siap, lakukan push file Anda ke GitHub dengan langkah-langkah berikut:

  1. Buka terminal atau command prompt, lalu navigasikan ke folder proyek.
  2. Inisialisasi Git di folder Anda:
    3. git init
  3. Tambahkan semua file proyek ke dalam repositori:
    4. git add .
  4. Buat commit pertama :
    5. git commit -m "Initial commit"
  5. Hubungkan repositori lokal dengan repositori GitHub :
    6. git remote add origin https://github.com/username/my-website.git
  6. Push Proyek ke GitHub :
    7. git push -u origin main

Langkah 4 : Aktivasi GitHub Pages

Langkah terakhir adalah mengaktifkan GitHub Pages di repositori Anda:

  1. Buka halaman repositori di GitHub, pilih Settings, lalu cari bagian Pages.
  2. Pada bagian Source, pilih main branch (atau master branch) dan klik Save.
  3. GitHub akan memproses dan menyediakan tautan situs web Anda. Anda bisa mengaksesnya di alamat: https://username.github.io/my-website/.
