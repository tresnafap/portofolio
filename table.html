Tutorial Membuat Table

Ketika membuat sebuah tabel, ada beberapa elemen dasar yang perlu diketahui. Elemen-elemen ini akan membantu agar tabel tertata dengan baik dan rapi di halaman web. Berikut elemen-elemen yang digunakan untuk membuat tabel sederhana pada HTML :

< table > < thead > < tbody > < tr > < th > < td >

Penjelasan Elemen :

< table > : Elemen ini merupakan elemen utama yang mendefinisikan tabel. Semua elemen tabel, seperti header, baris, dan sel, berada di dalam tag ini

< thead > : Elemen ini digunakan untuk mengelompokkan bagian kepala atau judul kolom dari tabel.

< tbody > : Elemen ini digunakan untuk mengelompokkan isi atau data dari tabel. Elemen ini digunakan untuk memisahkan isi tabel dari judul kolom yang ada di elemen thead.

< tr > : Elemen ini digunakan untuk mendefinisikan atau membuat baris dalam tabel.

< th > : Elemen ini digunakan untuk membuat sel header dalam tabel. Biasanya, elemen ini digunakan untuk mendefinisikan judul setiap kolom.

< td > : Elemen ini digunakan untuk mendefinisikan sel data dalam tabel. Setiap elemen ini merepresentasikan sebuah data di dalam tabel.


Langkah-Langkah Membuat Table Sederhana :

1. Membuat Struktur Dasar HTML

Sebelum membuat tabel, pastikan kamu sudah menulis struktur HTML yang benar. Dokumen HTML dimulai dengan deklarasi < !DOCTYPE html >, kemudian diikuti dengan elemen < html >, < head >, dan < body >.

2. Membuat Tabel Menggunakan Tag < table >

Tabel dalam HTML dibuat menggunakan tag < table >. Di dalam tag < table >, terdapat elemen-elemen lain yang menyusun baris, kolom, dan sel tabel. Elemen-elemen ini sudah dijelaskan diatas.

3. Mengisi Data kedalam Tabel

- Mengisi data judul kolom dilakukan di dalam elemen < th > yang berada di dalam elemen < thead >. Elemen ini digunakan untuk menampilkan judul setiap kolom, seperti "NIM", "Nama", "IPK", dan lainnya.
- Mengisi data di setiap kolom dilakukan di dalam elemen < td >. Elemen < td > digunakan untuk memasukkan nilai atau data yang sesuai dengan judul kolom, seperti NIM, Nama, IPK per semester, dan lain-lain.
- Mengganti baris data menggunakan elemen < tr >. Elemen < tr > ini berfungsi untuk membuat baris baru dalam tabel, sehingga kamu bisa memasukkan data mahasiswa berikutnya atau baris data tambahan.

4. Menerapkan Style CSS pada Tabel

Agar tampilan tabel lebih menarik dan mudah dibaca, kita bisa menambahkan CSS untuk mengatur desainnya. Dengan CSS, kamu bisa mengubah warna, jarak antar teks, hingga bentuk tabel.
Salah satu CSS yang penting adalah "border-collapse: collapse", CSS ini digunakan untuk menggabungkan antar cell
